 Quadruped Spider Robot

## Overview
This project involves the development of a four-legged (quadruped) spider robot capable of basic locomotion and balance. It was built as part of an academic project and presented during Smart India Hackathon (SIH) 2024.

## Features
- Coordinated movement of four legs  
- Basic walking and turning functionality  
- Stability-focused design  

## Components Used
- Servo motors  
- Microcontroller (e.g., Arduino/ESP-based)  
- Power supply  
- Mechanical frame  

## Working
The robot moves by coordinating the motion of its four legs using programmed motor control. Each leg follows a sequence that allows the robot to walk while maintaining balance.
